# Gsuke Keybindings

個人用のホットキーの設定集。
JIS配列キーボードを前提とする。

## 設定方法

* [Linux](linux/README.md)
* [Windows](windows/README.md)

## ホットキー一覧

| 入力キー        | 割り当てられる機能 |
| ----------- | --------- |
| 無変換         | IMEオフ     |
| 変換(※)       | IMEオン     |
| 変換(長押し)(※)  | 右Shift    |
| 無変換 + H     | ←         |
| 無変換 + J     | ↓         |
| 無変換 + K     | ↑         |
| 無変換 + L     | →         |
| 無変換 + A     | Home      |
| 無変換 + E     | End       |
| 無変換 + Space | Enter     |
| 無変換 + ;     | BackSpace |
| 無変換 + :     | Delete    |

※「カタカナひらがな」「右Shift」キーも「変換」キーと同じ動作となる。
